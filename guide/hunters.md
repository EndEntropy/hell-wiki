# 猎魔人

自从地狱入侵之后，地球的绝大多数地区都沦为废土。而在仅存的区域内，恶魔的威胁也不容小觑，因此**猎魔人**应运而生。

## 概述

猎魔人虽然初衷高贵，但实际上并不是什么光鲜的行业。里面绝大多数人都只是为了混一口饭吃，也很少有什么组织。事实上，猎魔人更接近于**雇佣兵**——现在的猎魔人除了"猎杀"恶魔，更多时候会接取其他委托，而且有相当一部分低阶猎魔人从未真正对付过恶魔。

## 分类

按力量的来源，猎魔人主要分为以下几类：

| 类型 | 所属聚集地 | 特点 |
|------|-----------|------|
| **灵体派** | 无特定 | 最原始的流派，凭自身灵体强行从地狱获取力量，极需天赋，易被恶魔盯上 |
| **义体猎魔人** | [赛博](/factions/cyber) | 使用义体机械，不惧痛觉和受伤 |
| **机械猎魔人** | [伦底纽姆](/factions/rondinium) | 使用圣化机械，精密而强大 |
| **强化猎魔人** | [塔帕姆](/factions/tapam) | 使用血肉强化技术，肉体极度强化 |
| **符咒猎魔人** | [长安](/factions/changan) | 使用符咒与丹药技术，东方风格 |
| **混合型** | 无特定 | 使用自研技术或将多种技术混用 |

## 生态

猎魔人的生态远不如人们想象的那样光鲜：

- 绝大多数猎魔人只是为了**谋生**
- 低阶猎魔人往往接取的是普通委托，而非真正的猎魔任务
- 很少有正式的**组织结构**，大多各自为战
- 高阶猎魔人才是真正具备对抗恶魔能力的人
