# 关于这个世界

欢迎来到**地狱降临**的世界观百科。

## 世界背景

自从**地狱入侵**之后，地球的绝大多数地区都沦为废土。人类退守到仅存的几个聚集地，在恶魔的威胁下艰难求存。

为了生存，人们发展出了如同神迹般的技术——每个聚集地都拥有独特的力量体系：

| 聚集地 | 核心技术 | 特征 |
|--------|---------|------|
| **长安** | 符咒 / 丹药 | 东方风格，层级分明 |
| **赛博** | 义体改造 | 科技驱动，金钱至上 |
| **伦底纽姆** | 圣化机械 | 精密工程，神秘祝福 |
| **塔帕姆** | 血肉强化 | 生物改造，肉体至上 |
