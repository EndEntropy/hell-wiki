# 圣化机械

> *"嗤……滋……"* 当你听到这些代表着蒸汽喷流和机械传动的声音时，你应该知道，那是圣化机械的声音。

圣化机械是 [伦底纽姆](/factions/rondinium) 的核心产业与技术，通过极其精密的结构和极其神秘的祝福，打造出强力的机械。

## 概述

使用圣化机械的机械猎魔人往往会搭配其他的技术一起使用，因为圣化机械更像是**武器**，而不是身体。

## 伤害方式

圣化机械造成伤害一般使用：

- **圣水蒸汽**射流
- **圣化金属蒸汽**射流

## 高阶能力

越往上，圣化机械的效果就越神秘：

| 等级 | 能力 |
|------|------|
| 基础 | 圣水蒸汽 / 圣化金属蒸汽射流 |
| 高级 | "收割"周围的时间，暂停或倒退 |
| 高级 | 压制地狱与灵体的力量，让战斗重归原始的力量比拼 |
| 高级 | 播放无法检测的心灵之声，直接感染敌人的精神和恶魔的存在 |

::: tip 核心特征
圣化机械最独特的地方在于它结合了**精密工程**与**神秘祝福**——纯粹的机械技术叠加了某种超自然的力量，这使得它在所有技术体系中独树一帜。
:::
