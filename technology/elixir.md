# 丹药

丹药是 [长安](/factions/changan) 的又一项独特技术，常以丸药、汤药、膏药等形式出现。

## 概述

丹药的效果各异，从修复身体伤势到强化各项能力，无所不包。

## 效果分类

| 类别 | 效果 |
|------|------|
| **治疗丹药** | 修复身体伤势 |
| **强化丹药** | 强化各项身体能力 |
| **未知丹药** | 据说冬司的人正在秘密研制具备**长生不老**功效的丹药 |

## 军事应用

长安的官方武装力量基本使用了丹药，[十二卫](/factions/changan#十二卫) 就是最典型的例子。

## 与其他技术的竞争

- 长安的**治疗丹药**与塔帕姆的**治疗技术**长期竞争
- 长安的**强化丹药**与塔帕姆的**强化纹身**互有优劣

::: tip 长生不老
据说冬司正在秘密研制的丹药具备使人生生不息、长生不老的功效。这一消息是否属实，仍是未知数。
:::
