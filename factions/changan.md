# 长安

> 人类的四个聚集地之一，拥有数量最多的人口。

长安位于陕西地区，无数巨大的建筑层层堆叠，形成如山一般的结构体。

## 概述

- **核心产业**：符咒技术
- **核心技术**：符咒 + 丹药
- **地位**：四大聚集地中人口最多，拥有最强的符咒工业

## 建筑

长安的城市结构呈**垂直堆叠**形态：

```
        ┌──────────┐
        │   岱     │  ← 首都（顶部）
        ├──────────┤
        │  九城圈   │  ← 内圈城市
        ├──────────┤
        │ 廿七城圈  │  ← 外圈城市
        ├──────────┤
        │   墟     │  ← 底部外围（非官方）
        └──────────┘
```

### 岱（首都）

长安的首都，位于结构体的顶部。岱内云雾缭绕，丹香四溢，天光明媚，实乃人间仙境。

- 居住着长安一半以上的**重要人物**和**高端战力**
- 六司的总部和**皇城**位于此
- **国子监**、**九重工坊**等顶尖机构坐落于此

### 三十六城

长安的城市除了首都岱，总数三十六。

**内圈九城：** 冀、兖、青、徐、扬、荆、豫、梁、雍

**外圈城市：** 丘、棘林、大穷、都广、焦侥、金丘、一目、积水以及其他城市

### 墟

长安最底部外围的区域，在**官方地图中不存在**。

- 生存环境好于外界废土，但也相当恶劣
- 秩序基本靠民间自发维持
- 居民包括：流浪者、被长安驱逐者、逃犯等
