# 名家

> 长安四流之一。执着于探索"名"与"实"之间的关系，在逻辑上的探索无人能及。

名家能有意识地寻找话语、规则的漏洞与不协调，并十分有效地利用这些漏洞与不协调，达成种种效果。

## 等级体系

| 等级 | 说明 |
|------|------|
| **学徒** | 入门等级 |
| **副导师** | 中级，对"名"与"实"有深入研究 |
| **导师** | 高级成员 |
| **教授** | 唯一，据说能够教导导师们的存在 |

## 内部派别

名家内部主要分为两派：

- **离坚白派** — 侧重于分析事物属性之间的独立性
- **合同异派** — 侧重于探讨事物之间的相同与差异

## 力量特点

名家的力量**诡异难缠**，总是频频出其不意。其特点如下：

- 善于发现话语与规则的**漏洞与不协调**，并加以利用
- 精于**逻辑分析**，推理能力极强
- **近身战斗能力较弱**——名家的优势不在正面对抗，而在于以奇制正

## 白马

名家掌控的独特空间与技术，是一个类似于客栈的地方。

**"白马"**的走廊遍布门，每一个门都对应现实的一处地方。只要在现实中设下一道通往"白马"的门，"白马"内部就必然会出现对应的门。

进入"白马"必须有人带领，内部**变幻莫测**，即便是学徒也偶有迷失。
