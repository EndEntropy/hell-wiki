# 儒家

> 长安四流之一，有时也被认为是四流之首。以"礼"为核心理念，重视礼法与秩序。

儒家十分重视礼法，尽管不会对失礼之人大打出手，但也绝不会给其好脸色。

## 等级体系

儒家内部有一套独特的等级体系，成员被分为四个等级：

| 等级 | 说明 |
|------|------|
| **儒生** | 入门等级，需要学会运用"礼"的力量 |
| **儒士** | 在"礼"的理解与践行上更进一步 |
| **贤人** | 在整个长安仅有七十二位，被统称为**七十二贤人** |
| **圣人** | 最高等级 |

加入儒家首先要学会运用"礼"的力量，随着读书与理解"礼"、践行"礼"，儒家成员的力量就会逐步提升，等级也会随之增高，受到其他成员的尊重。

尽管儒家的变强方法看起来十分简单，但却需要拥有**极高的悟性**，一般人哪怕是儒生也无法企及。

## "礼"

儒家的核心力量。

理解"礼"、践行"礼"就会从多方面获得强化：

- **身体素质**提升
- **灵魂强度**增强
- **地狱抗性**提高
- **恶魔特攻**加成

如果不践行乃至违背"礼"，就会快速变弱，乃至回到最开始的状态。

### 叠加效应

大量的儒家成员聚集在一起，他们的"礼"就会**互相叠加**，达到一个恐怖的地步。这使得儒家的集体力量远超个体之和。

## 科举

儒家会定期举行名为"科举"的考试，成员自愿参加，主要是考核"礼"。

| 名次 | 称号 |
|------|------|
| 第一名 | **状元** |
| 第二名 | **榜眼** |
| 第三名 | **探花** |

尽管科举并没有什么实质上的提升，但儒家成员往往十分追求这份名号，因为它是他们"礼"的证明。

### 与长安政府的关系

科举与长安政府并无直接联系。不过，毕竟能在科举中拿个好名次的儒家成员一定足够有礼仪，因此长安政府还是比较乐意录用这些人的。但也会保持谨慎——毕竟长安政府有时会做一些在儒家看来违反"礼"的事情。
