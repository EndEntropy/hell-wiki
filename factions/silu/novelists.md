# 小说家

> 长安四流之一。最开始只是一群执着于用文字捕捉现实的人聚集在一起，后来说书人、诗人等也陆续加入，成为长安文字艺术行业毋庸置疑的权威之一。

小说家并没有什么崇高的理想，这只是一群人为了实现艺术抱负而形成的组织，因此成员的道德水平参差不齐。

## 等级体系

| 等级 | 说明 |
|------|------|
| **学徒** | 入门等级 |
| **天王** | 中级成员 |
| **大神** | 高级成员 |
| **白金** | 顶级，是否存在更上的等级尚未可知 |

## 内部派别

小说家内部分为许多派别：

| 派别 | 特征 |
|------|------|
| **乡野派** | 原教旨主义，只采集民间故事 |
| **悬疑派** | 讲述灵异故事 |
| **星火派** | 讲述反抗故事 |
| **审查派** | 以**河蟹**作为徽标，在书库里巡逻，驱逐与杀死擅闯的人 |

## 道德光谱

小说家的成员道德水平参差不齐，从圣人到疯子皆有：

- 可以为普通人的生命而放弃即将完成的伟大作品的圣人
- 干脆**制造苦难**以获得创作素材和灵感的疯子

## 书库

小说家的重要据点，每个城市都设有数个书库。

实际上所有的书库都是**联通且一体的**，里面收录了小说家的成员们上传的作品。小说家的活动多数在书库里举行。

书库也是**审查派**的据点——他们以河蟹作为徽标，在书库里巡逻，驱逐与杀死擅闯的人。
