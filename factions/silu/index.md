# 四流

> 长安内部四个庞大的势力，体量之巨令长安政府也无法轻易出手。

四流是长安城内独立于六司行政体系之外的四大民间势力。它们各自拥有庞大的成员基础、独特的力量体系和内部等级制度，在长安的社会格局中扮演着举足轻重的角色。

## 概述

四流之所以被称为"流"，是因为它们不像六司那样属于官方行政机构，而是以**理念与传承**为纽带凝聚起来的力量。尽管如此，四流的体量之大、影响力之深，使得即便是长安政府也必须慎重对待。

## 与长安政府的关系

四流并非官方机构，不受六司直接管辖。但由于其庞大的体量和影响力，长安政府在处理与四流的关系时必须谨慎：

- 四流成员不受六司的直接指挥
- 长安政府在需要时可能与四流合作
- 四流之间的冲突也可能牵涉到长安政府的利益
