# 六司

长安的六大行政机构，分别是**天司、地司、春司、夏司、秋司、冬司**。

## 天司

- **地位**：六司之首，统御六司
- **职责**：考核管理、猎魔人管理、财政统筹
- **最高长官**：大冢宰
- **现任**：**毕旦**

## 地司

- **地位**：六司之一
- **职责**：安邦富民、管理城市与赋税、商贸与教育
- **最高长官**：大司徒
- **现任**：**扶德**

::: tip 工坊管理
所有工坊都需要在地司备案，取得工坊证书。详见 [工坊体系](/factions/changan#工坊)。
:::

## 春司

- **地位**：六司之一
- **职责**：典乐掌教、占卜未来、外交礼仪
- **最高长官**：大宗伯
- **现任**：**伯昂**

## 夏司

- **地位**：六司之一
- **职责**：掌军统兵、武官选拔、拱卫长安
- **最高长官**：大司马
- **现任**：**子光**

## 秋司

- **地位**：六司之一
- **职责**：掌禁执法、审判刑狱、社会治安
- **最高长官**：大司寇
- **现任**：**未知**

::: warning 信息被隐藏
秋司大司寇的信息被刻意隐藏，其身份和背景是长安的未解之谜之一。
:::

## 冬司

- **地位**：六司之一
- **职责**：务本兴工，专利审核、工程建设、武器制造
- **最高长官**：大司空
- **现任**：**鲧**

::: info 秘密项目
据说冬司正在秘密研制的 [丹药](/technology/elixir) 具备长生不老的功效。
:::
